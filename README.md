# DS3Tool
Dark Souls 3 Tool originally developed by https://github.com/kh0nsu 

This targets game version 1.15 only, **not** 1.15.2.

Included features:
* No Death
* Item Gib
* Mass Item Gib
* One-shot Mode
* Disable Enemy AI
* Enemy Repeat Last Action
* Hitbox view
* Event View
* No Clip
* Save/Load Position
* Character Mesh
* Freeze Enemy HP
* Cinder Phase Control
